# Loremaster

A local-first D&D rules assistant. Ask rules questions and get answers grounded in
your own rulebook PDFs — runs entirely on your machine, no cloud, no subscriptions.

## Download

Grab the latest installer from **[Releases](../../releases/latest)** →
download `Loremaster-Setup-*.exe` and run it.

- If Windows shows a blue "protected your PC" box: **More info → Run anyway**
  (the app is unsigned — this is expected).
- First launch downloads the AI models (~9 GB) — be on good Wi-Fi. After that,
  it runs fully offline.

## Updating

The app updates itself — new releases are picked up automatically on launch.
